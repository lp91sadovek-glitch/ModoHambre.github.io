# Modo Hambre

## ¿Qué es Modo Hambre?

Modo Hambre es una web pensada para las pequeñas empresas gastronómicas, ya que las plataformas de delivery actuales les cobran entre un 40% y un 60% de comisión por cada pedido realizado. Esto provoca que el gastronómico deba aumentar el valor final de su producto, acotando sus ventas y sus posibilidades de alcanzar a nuevos comensales por recomendación.

---

## ¿Cómo funciona?

Su función es simple: 
1. El usuario tiene diferentes opciones de platos y selecciona el o los que desea.
2. Automáticamente se agregan a un carrito de compras.
3. En la misma ventana del carrito se muestra el valor total del producto.
4. Una vez confirmado el pedido, se envía un mensaje a través de WhatsApp. 

De esta forma, la coordinación del pago y el envío se realiza directamente entre el cliente y el comercio.
